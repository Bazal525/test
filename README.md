## Opis projektu
Projekt "Kino" to prosty system zarządzania filmami oraz rezerwacjami w kinie.  
Funkcjonalności aplikacji obejmują:
- Dodawanie nowych filmów (tytuł, opis, godzina seansu).
- Przeglądanie dostępnych filmów.
- Rezerwowanie biletów na wybrane seanse.
- Wyświetlanie listy dokonanych rezerwacji.

Aplikacja została zbudowana w oparciu o framework Flask i SQLite jako bazę danych. Dodatkowo zawiera zaimplementowane testy jednostkowe oraz integracyjne.
